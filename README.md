## Installation

```shell
$ npm install translit-geo
```

## Usage

```js
import translit from 'translit-geo'

translit('abgdevzTiklmnopJrstufqRySCcZwWxjh');
=> 'აბგდევზთიკლმნოპჟრსტუფქღყშჩცძწჭხჯჰ'
```
